# the-Recipe-book

Full-Stack JS engineer test assessment

## Installing

### For client:

1. `cd FE`
2. `npm install`

### For server:

1. `cd BE`
2. `npm install`

## Runnin the app

### To run the server:

While in BE directory
`npm start`

### To run the client:

While in FE directory
`npm run dev`
